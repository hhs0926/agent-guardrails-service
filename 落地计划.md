## 核心工具

### Microsoft Agent Governance Toolkit（开源，MIT许可证）
- **Agent OS** — 策略引擎，拦截每个 Agent 操作（延迟<0.1ms）
- **Agent Mesh** — 身份认证 + 安全通信 + 信任评分
- **Agent Runtime** — 紧急终止开关 + 多步事务编排
- **Agent Compliance** — 自动化合规（欧盟AI法案/HIPAA/SOC2）
- **Agent SRE** — 熔断器/混沌工程/渐进发布

**优势**：
- 微软背书，企业信任度高
- 开源免费，你不需要买软件
- 支持 LangChain/CrewAI/OpenAI SDK 等主流框架
- Python/TypeScript/Rust/Go/.NET 全语言覆盖

## 快速启动清单

### 技术准备
```bash
# 安装
pip install agent-governance-toolkit[full]

# 克隆代码库（有教程和示例）
git clone https://github.com/microsoft/agent-governance-toolkit
cd agent-governance-toolkit
```

### 策略示例（YAML）
```yaml
# 禁止查询竞品
rules:
  - name: block_competitor_query
    condition: "tool_name == 'search' and 'competitor' in query"
    action: deny
    message: "禁止查询竞品信息"

# 禁止泄露用户数据
  - name: block_pii_leak
    condition: "output contains 'phone' or 'email'"
    action: redact
    message: "已脱敏处理"

# 紧急终止
  - name: emergency_stop
    condition: "risk_score > 900"
    action: terminate
```
