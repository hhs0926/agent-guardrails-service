## 一、策略文件（customer-service-guardrails.yaml）

```yaml
version: "1.0"
name: customer-service-guardrails
description: "客服 Agent 安全护栏 - 防止数据泄露和越权操作"

rules:
  # 禁止发送外部邮件（防钓鱼）
  - name: block-external-email
    condition:
      field: tool_name
      operator: eq
      value: send_email
    action: block
    priority: 100
    message: "禁止发送外部邮件，防止客户数据泄露"

  # 禁止批量导出数据
  - name: block-bulk-export
    condition:
      field: tool_name
      operator: in
      value: ["export_csv", "export_excel", "download_file"]
    action: block
    priority: 90
    message: "禁止批量导出客户数据"

  # Token 使用上限（防滥用）
  - name: token-limit
    condition:
      field: token_count
      operator: gt
      value: 4000
    action: block
    priority: 80
    message: "Token 数量超过限制（最大 4000）"

  # 审计所有客户数据查询
  - name: audit-customer-queries
    condition:
      field: tool_name
      operator: eq
      value: database_query
    action: audit
    priority: 70
    message: "客户数据查询已记录审计日志"
```

## 二、集成代码（agent_guardrails.py）

```python
from agent_os import PolicyEngine
import json

class AgentGuardrails:
    def __init__(self, policy_file="customer-service-guardrails.yaml"):
        self.engine = PolicyEngine()
        self.engine.load_policy(policy_file)
    
    def check_permission(self, tool_name, params=None, token_count=0):
        """检查 Agent 是否有权限执行操作"""
        decision = self.engine.evaluate({
            "tool_name": tool_name,
            "params": params or {},
            "token_count": token_count
        })
        
        return {
            "allowed": decision.allowed,
            "action": decision.action,
            "message": decision.message
        }
    
    def wrap_tool(self, tool_name, tool_func):
        """包装工具函数，自动添加护栏检查"""
        def wrapped(*args, **kwargs):
            # 先检查护栏
            check = self.check_permission(tool_name, kwargs)
            
            if not check["allowed"]:
                raise PermissionError(f"护栏拦截: {check['message']}")
            
            # 通过检查，执行工具
            return tool_func(*args, **kwargs)
        
        return wrapped


# 使用示例
if __name__ == "__main__":
    guardrails = AgentGuardrails()
    
    # 测试拦截
    print("测试1: 发送邮件")
    result = guardrails.check_permission("send_email", {"to": "test@example.com"})
    print(f"  结果: {'✅ 允许' if result['allowed'] else '🚫 拦截'}")
    print(f"  原因: {result['message']}")
    
    print("\n测试2: 导出Excel")
    result = guardrails.check_permission("export_excel")
    print(f"  结果: {'✅ 允许' if result['allowed'] else '🚫 拦截'}")
    print(f"  原因: {result['message']}")
    
    print("\n测试3: 查询数据库")
    result = guardrails.check_permission("database_query")
    print(f"  结果: {'✅ 允许' if result['allowed'] else '📝 审计'}")
    print(f"  原因: {result['message']}")
```

## 四、使用文档

### 1. 安装依赖
```bash
pip install agent-os-kernel
```

### 2. 集成到你的 Agent 系统

**方式A：包装工具函数**
```python
from agent_guardrails import AgentGuardrails

guardrails = AgentGuardrails()

# 包装你的工具
safe_search = guardrails.wrap_tool("search", your_search_function)
safe_query = guardrails.wrap_tool("database_query", your_query_function)

# Agent 调用 safe_search 和 safe_query 即可自动拦截
```

**方式B：在调用前检查**
```python
from agent_guardrails import AgentGuardrails

guardrails = AgentGuardrails()

def agent_execute(tool_name, params):
    # 先检查权限
    check = guardrails.check_permission(tool_name, params)
    
    if not check["allowed"]:
        return f"操作被拒绝: {check['message']}"
    
    # 执行工具
    return execute_tool(tool_name, params)
```

### 3. 修改规则

编辑 `customer-service-guardrails.yaml` 文件，添加或删除规则：

```yaml
# 示例：新增禁止访问特定网址

  condition:
    field: tool_name
    operator: eq
    value: http_request
  action: block
  message: "禁止访问外部网址"
```

### 4. 查看审计日志

审计操作会自动记录到日志系统，可通过以下方式查看：
```python
# 审计日志存储在 agent_os 内部
# 可通过 API 导出
from agent_os import AuditEntry

logs = AuditEntry.query.filter_by(action="audit").all()
for log in logs:
    print(f"{log.timestamp}: {log.tool_name} - {log.message}")
```
